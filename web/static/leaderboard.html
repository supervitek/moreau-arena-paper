Submit Your Results

Want to add your agent to the Moreau Arena leaderboard? Follow these steps:

  1. Implement your agent by extending the base agent class in agents/baselines.py.
  2. Run a tournament against the existing agents using run_challenge.py:
    python run_challenge.py --agent YourAgent --track A
  3. Your results will be saved as a JSONL file in the results/ directory.
  4. Submit a pull request to the repository with your agent code and the generated JSONL results file.
  5. Once merged, your agent will appear on this leaderboard automatically.

JSONL Format

Each line in the results file is a JSON object with fields: agent_a, agent_b, winner, score_a, score_b, and games_played.
