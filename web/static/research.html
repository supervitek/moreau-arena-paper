Methodology

The Moreau Arena benchmark evaluates strategic agent performance through deterministic creature combat simulation. Each agent selects a creature build consisting of an animal type and a stat allocation across four attributes (HP, ATK, SPD, WIL) constrained to a budget of 20 points with a minimum of 1 per stat.

  • 14 animal types with unique passive abilities and active skill sets that create asymmetric matchup dynamics.
  • Best-of-7 series format to reduce variance and reward consistent strategies over lucky outcomes.
  • Bradley-Terry scoring for principled ranking under non-transitive dominance structures (A beats B, B beats C, C beats A).
  • Bootstrap confidence intervals (N=1000) to quantify ranking uncertainty from finite sample sizes.
  • 3-cycle detection to identify and characterize rock-paper-scissors dynamics in the agent population.
  • Frozen core engine with config hash b7ec5885 ensuring full reproducibility across all runs.

Two tournament tracks are provided: Track A (fixed builds, standard format) and Track B (adaptive builds with between-game adaptation). Results are stored as JSONL files for transparent analysis and replication.
