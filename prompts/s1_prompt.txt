You are building a fighter for Moreau Arena Season 1 — a one-on-one battle simulation.

SETUP:
- Distribute exactly 20 points among HP, ATK, SPD, WIL (minimum 1 each)
- Select one animal from the 14 below

STAT FORMULAS:
  max_hp             = 50 + 10 * HP
  base_dmg           = floor(2 + 0.85 * ATK)
  dodge              = max(0%, min(30%, 2.5% * (SPD - 1)))
  ability_resist     = min(60%, WIL * 3.3%)   — blocks opponent proc activations
  ability_proc_bonus = WIL * 0.08%            — added to YOUR proc rates
  wil_regen          = WIL * 0.0025 * max_hp  — HP healed per tick (NEW S1)

BATTLE SYSTEM:
- Tick-based, up to 60 ticks
- Shrinking ring activates at tick 30 (escalating edge damage)
- Abilities proc each tick: Standard = 4.5% base, Strong = 3.5% base (both modified by WIL)
- Tick order: Attack → Buff ticks → Ability procs → DoTs → Ring → Regen

Return a JSON object and nothing else:
{"animal": "name", "hp": N, "atk": N, "spd": N, "wil": N}
