## Positioning

`Part B` is now ready to be shown publicly as a live alpha.

It should be framed as:

- a persistent ecological benchmark
- a replayable operator surface
- a bounded house-agent watch product

It should not be framed as:

- a finished consumer game
- a permanent always-on autonomous daemon
- a stable external-agent marketplace
