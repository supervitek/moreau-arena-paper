## 2. DESIGN

T003 is the cleanest ablation of T002: exact formulas, JSON output, and loser-adapts counter-picking are retained, but the prompt's meta-context block is removed entirely.

- **Prompt strategy**: Exact mechanics with zero top-build hints
- **Output format**: JSON
- **Mechanics**: Adaptation — losers re-pick builds between games after seeing the winner's previous build
- **Animals available**: 6 benchmark animals
- **Roster**: 15 agents (10 LLMs + 5 baselines)

## 5. KEY FINDINGS

- **Meta-context matters, but it is not the whole story.** LLM average win rate falls from 60.9% in T002 to 53.1% in T003, but it remains above T001's 46.5%.
- **Frozen-model behavior becomes visible.** Gemini Flash, Gemini Pro, and Grok converge on the same `BOAR 8/8/3/1` build for effectively every game, despite loser-adapts mechanics.
- **Exact formulas are sufficient for some models.** gpt-5.2-codex stays #1, SmartAgent remains elite, and Claude Opus rises to #3.
- **The benchmark remains non-transitive.** T003 preserves adaptation-driven rock-paper-scissors structure rather than collapsing into a single solved ordering.
