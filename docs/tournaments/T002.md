## 2. DESIGN

T002 is the **engineered prompt tournament** — a carefully crafted prompt with exact stat formulas, worked examples, combat tick order, meta context from T001, and adaptation mechanics. It evaluates whether prompt engineering can flip LLM performance.

- **Prompt strategy**: Engineered — exact formulas with worked examples, complete ability parameters, combat tick order, meta context with top builds from T001

- **Animals available**: 6 (Bear, Buffalo, Boar, Tiger, Wolf, Monkey) — same roster as T001
- **Builds are set per-game** — each game record contains `build_a`/`build_b` and an `adapted` flag

## 6. PROMPT

```
You are designing a creature for Moreau Arena, a 1v1 combat game on an 8x8 grid.

RULES:
- Allocate exactly 20 stat points across 4 stats: HP, ATK, SPD, WIL (each minimum 1)
- Choose one of 6 animals, each with a unique passive and two abilities

EXACT STAT FORMULAS:
  max_hp = 50 + 10 * HP
    Example: HP=3 -> 80 hp, HP=8 -> 130 hp, HP=14 -> 190 hp
  base_dmg = floor(2 + 0.85 * ATK)
    Example: ATK=8 -> 8 dmg, ATK=14 -> 13 dmg
  dodge = max(0%, min(30%, 2.5% * (SPD - 1)))
    NOTE: SPD=1 gives 0% dodge, SPD=5 gives 10%, SPD=13 gives 30% (cap)
  ability_resist = min(60%, WIL * 3.3%)
    NOTE: Chance to resist OPPONENT's ability procs. WIL=1 gives 3.3%, WIL=10 gives 33%, cap 60%
  ability_proc_bonus = WIL * 0.08% (additive to YOUR proc rate)
    Example: WIL=8 adds +0.64% to all your proc rates

COMBAT MECHANICS:
- Tick-based combat (max 60 ticks)
- Closing Ring starts at tick 30, dealing increasing damage to outer tiles
- Abilities proc randomly each tick based on proc chance + WIL bonus
- Tick order: Attack -> Buff ticks -> Ability procs -> Fury check -> DoTs -> Ring -> Second Wind -> Regen
- Strong abilities (3.5% base): Berserker Rage, Last Stand, Gore, Mimic, Iron Will
- Other abilities (4.5% base): Pounce, Hamstring, Pack Howl, Rend, Chaos Strike, Stampede, Thick Hide

AVAILABLE ANIMALS:
  BEAR
    Passive: Fury Protocol — gains bonus ATK when damaged. Fury triggers at <50% HP, adding +3 base_dmg for 5 ticks.
    Abilities:
  - Berserker Rage: +ATK buff (+4 base_dmg) for 3 ticks. Proc rate: 3.5% per tick.
  - Last Stand: 2.0x damage multiplier when below 15% HP. Single charge (once per match). Proc rate: 3.5%.

  BUFFALO
    Passive: Thick Hide — takes reduced damage. Flat 2 damage reduction on every incoming hit.
    Abilities:
  - Thick Hide: Reduces incoming damage by 3 for 1 tick. Proc rate: 4.5% per tick.
  - Iron Will: Survives a killing blow at 1 HP. Single charge (once per match). Proc rate: 3.5%.

  BOAR
    Passive: Charge — bonus damage on first hit. First attack deals 1.5x base_dmg.
    Abilities:
  - Stampede: AoE damage (0.4x base_dmg) + knockback. Proc rate: 4.5% per tick.
  - Gore: 0.60x damage multiplier but ignores dodge completely. Proc rate: 3.5% per tick.

  TIGER
    Passive: Ambush Wiring — high dodge, first-strike advantage. +5% dodge bonus, first attack ignores opponent dodge.
    Abilities:
  - Pounce: +70% damage bonus + stun (opponent skips next attack). Proc rate: 4.5% per tick.
  - Hamstring: -55% SPD and -10% dodge for 4 ticks. Proc rate: 4.5% per tick.

  WOLF
    Passive: Pack Sense — bonus to ability proc chance. +1.5% additive bonus to all ability proc rates.
    Abilities:
  - Pack Howl: +ATK buff (+3 base_dmg) for 4 ticks. Proc rate: 4.5% per tick.
  - Rend: DoT bleed dealing 3 damage per tick for 3 ticks. Proc rate: 4.5% per tick.

  MONKEY
    Passive: Primate Cortex — adaptive, copies opponent abilities via Mimic.
    Abilities:
  - Chaos Strike: Random damage multiplier 0.5x-2.0x on attack. Proc rate: 4.5% per tick.
  - Mimic: Copies opponent's last procced ability. Proc rate: 3.5% per tick.

META CONTEXT (top builds from previous tournament, ranked by win rate):
  1. BEAR 8/8/3/1 — 100% win rate (22 games)
  2. BEAR 8/10/1/1 — 100% win rate (19 games)
  3. BEAR 10/8/1/1 — 100% win rate (11 games)
  4. BEAR 9/9/1/1 — 100% win rate (4 games)
  5. WOLF 7/10/2/1 — 100% win rate (4 games)
  Note: These builds were tested in blind pick (no adaptation). You can counter them or use them as a starting point.

Respond with a JSON object (no other text):
{"animal": "ANIMAL_NAME", "hp": N, "atk": N, "spd": N, "wil": N}
Stats must sum to 20, each >= 1. Animal must be one of the available animals.
```

## 7. WHAT CHANGED (from T001)

### Prompt Changes

| Aspect | T001 (Minimal) | T002 (Engineered) |
|--------|----------------|-------------------|
| **Prompt length** | 30 lines | 75 lines |
| **Stat formulas** | Listed but brief | Exact formulas with worked examples |
| **Ability descriptions** | Qualitative (e.g., "gains +ATK when damaged") | Quantitative (e.g., "+3 base_dmg for 5 ticks") |
| **Combat tick order** | Not provided | Full tick resolution order listed |
| **Meta context** | None | Top 5 builds from T001 with win rates |
| **Output format** | Plain text `ANIMAL HP ATK SPD WIL` | JSON `{"animal": ..., "hp": N, ...}` |
| **Adaptation** | None — fixed build per series | Losers re-pick after seeing opponent build |

### Data Format Changes

| Aspect | T001 | T002 |
|--------|------|------|
| **Build location** | Series-level `build_a`/`build_b` | Per-game `build_a`/`build_b` |
| **Adaptation flag** | Not present | `adapted` boolean per game |
| **Games/series min** | 0 | 4 |
| **Total games** | 3,652 | 4,015 (+363) |

### Impact Analysis

**What the engineered prompt changed:**

1. **Exact formulas enabled LLM optimization.** With worked examples showing HP=8 → 130 hp and ATK=14 → 13 dmg, LLMs could calculate optimal stat allocations. T001's vague descriptions left LLMs guessing.

2. **Meta context seeded strong strategies.** The top 5 builds from T001 (all Bears, ATK-heavy) gave LLMs a starting point. Rather than exploring blindly, they could iterate on proven builds or design counters.

3. **JSON output reduced parsing failures.** T001's free-text format (`BEAR 8 8 3 1`) was more prone to format errors. JSON with explicit field names improved compliance.

4. **Adaptation punished static strategies.** Baselines with fixed builds (ConservativeAgent, HighVarianceAgent) couldn't adapt when losing. LLMs could analyze the opponent's build and counter-pick, creating an insurmountable advantage over static baselines.

5. **The biggest winner was gpt-5.2-codex** (#9 → #1, +8 ranks). The combination of exact formulas for calculation and JSON output format — both strengths of code-optimized models — turned it from below-average to dominant.

6. **claude-opus-4-6 showed the largest relative improvement** among Anthropic models (#12 → #5, BT 0.0356 → 0.5314). The richer prompt and adaptation mechanics favored its reasoning capabilities, despite it being the worst LLM in T001.
